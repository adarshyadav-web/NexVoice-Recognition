# NexVoice Recognition

A premium AI-powered Speech-to-Text application using the Web Speech API with a modern HUD aesthetic.

## 🚀 Features

- **Real-time Transcription**: Converts voice to text instantly.
- **Pulsing Mic HUD**: Visual feedback when the AI is listening.
- **Clipboard Integration**: One-click copy of transcribed text.
- **Modern Dark UI**: High-impact design for productivity.

## 🛠️ Technologies Used

- **Web Speech API**
- **HTML5**, **CSS3**, **JavaScript**
